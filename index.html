Welcome to RESTful RZNU project site.

Base url

    localhost:8080

Base url path

    /api

Accounts /accounts</>

    @GET (userId:long) return Account - account
    @POST(input:Account) return Account - created account
    @PUT(input:Account) return Account - updated account
    @DELETE(userId:long) return HttpStatus
    @GET() retrun Collection - get all accounts

Images /{userId}/images</>

    @GET(userId:long) return Collection - images from user with id=userId
    @GET(userId:long,imageId:long) return Image - image with id=imageId and from user with id=userId
    @POST(userId:long,input:Image) return HttpStatus - add image input to user with id=userId
    @DELETE(userId:long,imageId:long) return HttpStatus - remove image with id=imageId from user with id=userId

Models

Account

    List images
    Long id
    String username
    String password

Image

    Long id
    String name
